# PlayMCP MCP 서버 등록 가이드

## 1. PlayMCP란?

PlayMCP는 카카오에서 운영하는 **MCP(Model Context Protocol) 기반 개방형 플랫폼**입니다.

- 웹 브라우저에서 바로 사용 가능
- 카카오 계정으로 로그인
- 개발자가 만든 MCP 서버를 등록하고 공개 가능
- 다른 사용자들이 도구함에 담아서 Claude, ChatGPT 등에서 사용

**공식 사이트**: https://playmcp.kakao.com/

---

## 2. 등록 요구사항

### 기술 스펙

| 항목 | 요구사항 |
|------|----------|
| 프로토콜 | **Streamable HTTP만** 지원 |
| stdio | ❌ 지원 안함 |
| SSE | ❌ 지원 안함 |
| 서버 | 직접 호스팅 필요 |
| 도메인 | **필수** (IP 주소 불가) |
| HTTPS | 필수 |

### 기타 제약사항

- 설명 글자 수 제한: **20,000자**
- 서버 운영 비용 자부담
- 등록 후 내용 변경 시 **재심사** 필요

---

## 3. 등록 프로세스

### Step 1: MCP 서버 개발
- Streamable HTTP 방식으로 MCP 서버 구현
- 필수 메서드: `initialize`, `tools/list`, `tools/call`
- JSON-RPC 2.0 형식

### Step 2: 서버 호스팅
- 서버 배포 (AWS, Vercel, Railway 등)
- **도메인 설정 필수** (IP 주소 등록 불가)
- HTTPS 적용

### Step 3: PlayMCP 개발자 콘솔 등록
1. https://playmcp.kakao.com 접속
2. 카카오 계정 로그인
3. **개발자 콘솔** 메뉴 진입
4. MCP 서버 정보 입력:
   - 서버 이름
   - 설명 (20,000자 제한)
   - 엔드포인트 URL (도메인)
   - 인증 방식 (Key/Token 또는 OAuth)
   - 도구(Tool) 정보

### Step 4: 심사 대기
- PlayMCP는 **심사제**
- 심사 기준: 서비스 안정성, 편의성, 창의성
- 심사 통과까지 대기

### Step 5: 전체 공개 전환
- 심사 완료 후 최초 상태는 **'나에게만 공개'**
- **전체 공개**로 직접 변경 필요

### Step 6: 응모 (공모전 참여 시)
- 전체 공개 전환 후
- 공모전 페이지에서 응모 버튼 클릭
- 1회만 응모 가능

---

## 4. 심사 기준

PlayMCP 심사 시 평가 항목:

1. **서비스 안정성**
   - 서버 가동률
   - 에러 처리
   - 응답 속도

2. **편의성**
   - 사용하기 쉬운 도구 설계
   - 명확한 설명

3. **창의성**
   - 독창적인 아이디어
   - 실용적인 활용 가치

---

## 5. 주의사항

### 심사 관련
- 등록 당시 설명과 달라지면 **재심사** 필요
- 도구 개수 변경 시에도 재심사
- 허위 정보 등록 시 자동 취소

### 기술 관련
- stdio, SSE 방식 서버는 등록 불가
- IP 주소로는 등록 불가 (도메인 필수)
- 서버 다운 시 사용자들이 이용 불가

### 비용 관련
- 서버 호스팅 비용 자부담
- 무료 옵션: Vercel, Railway (제한적), ngrok 무료 도메인
